📦 CSS Box Model

Основи блокової моделі CSS - фундамент веб-верстки

Що таке Box Model?

CSS Box Model - це концептуальна модель, яка описує, як елементи HTML відображаються на веб-сторінці. Кожен елемент складається з чотирьох основних частин:

MARGIN (зовнішній відступ)
BORDER (рамка)
PADDING (внутрішній відступ)
CONTENT
(вміст)

Компоненти Box Model

Компонент Опис CSS властивість
Content Фактичний вміст елемента (текст, зображення) width, height
Padding Внутрішній відступ між вмістом і рамкою padding
Border Рамка навколо padding і content border
Margin Зовнішній відступ від інших елементів margin

Інтерактивна демонстрація

Box Model
.demo-box { margin: 20px; border: 5px solid #c53030; padding: 20px; width: 200px; height: 100px; }

box-sizing: властивість

💡 Важливо: Властивість box-sizing визначає, як обчислюється загальний розмір елемента.
Значення Опис Формула розрахунку
content-box (за замовчуванням) Width/height стосується тільки вмісту Total = width + padding + border + margin
border-box Width/height включає padding і border Total = width + margin
/* Сучасний підхід - border-box для всіх елементів */ * { box-sizing: border-box; } /* Або тільки для конкретного елемента */ .container { box-sizing: border-box; width: 300px; padding: 20px; border: 2px solid #333; /* Загальна ширина залишається 300px */ }

Margin Collapsing (Злиття відступів)

⚠️ Увага: Вертикальні margin'и сусідніх елементів можуть зливатися в один!

Коли відбувається злиття:

  • Між двома блоковими елементами один над одним
  • Між батьківським і дочірнім елементом
  • У порожніх елементів
/* Приклад margin collapsing */ .box1 { margin-bottom: 30px; } .box2 { margin-top: 20px; } /* Результат: між box1 і box2 буде тільки 30px відступу, а не 50px! */ /* Як уникнути: */ .prevent-collapse { padding: 1px 0; /* або border: 1px solid transparent; */ }

Практичні поради

🎯 Кращі практики:
  • Використовуйте box-sizing: border-box для всіх елементів
  • Для layout'а краще використовувати Flexbox або Grid
  • Уникайте фіксованих розмірів - використовуйте відносні одиниці
  • Пам'ятайте про margin collapsing при верстці
❌ Поширені помилки:
  • Забування про padding і border при розрахунку розмірів
  • Неврахування margin collapsing
  • Використання негативних margin'ів без розуміння наслідків

Приклади використання

/* Центрування блоку */ .center-block { width: 300px; margin: 0 auto; /* горизонтальне центрування */ } /* Рівні відступи */ .card { padding: 1rem 1.5rem; /* вертикальний | горизонтальний */ margin: 1rem 0; /* відступи зверху і знизу */ border: 1px solid #ddd; border-radius: 8px; } /* Reset для списків */ ul, ol { margin: 0; padding: 0; list-style: none; } /* Responsive padding */ .container { padding: 1rem; } @media (min-width: 768px) { .container { padding: 2rem; } }
