📐 Розміри та відстані

📏 Інтерактивна демонстрація типографічних відстаней

Типографіка - це мистецтво та техніка організації тексту таким чином, щоб зробити письмову мову читабельною та привабливою при відображенні.

Правильне налаштування відстаней між літерами, словами та рядками значно покращує сприйняття тексту користувачем.

📊 Рекомендовані співвідношення

🎯 Золоті правила типографіки:
  • Line-height: 1.4-1.6 для основного тексту, 1.2-1.3 для заголовків
  • Довжина рядка: 45-75 символів (оптимально 65)
  • Розмір шрифту: мінімум 16px для основного тексту
  • Контраст: мінімум 4.5:1 для звичайного тексту

Для технічних блогів та документації
🎯 Правила поєднання шрифтів:
  • Максимум 2-3 шрифти на одній сторінці
  • Контраст стилів: serif + sans-serif або display + text
  • Спільні пропорції: схожа висота x-height
  • Ієрархія: декоративні тільки для заголовків
  • Мета дизайну: шрифти повинні підтримувати загальний стиль
🎨 Коли використовувати декоративні шрифти:
  • Logotype та брендинг - створення унікальної ідентичності
  • Заголовки і акценти - не більше 1-2 рядків
  • Тематичні проекти - відповідність стилю контенту
  • Call-to-action - привернення уваги до ключових елементів
  • Великі розміри - мінімум 24px для збереження читабельності
⚠️ Коли НЕ використовувати декоративні шрифти:
  • Довгі тексти - швидко втомлюють очі
  • Малі розміри - втрачають читабельність
  • Форми введення - користувач повинен легко читати свої дані
  • Мобільні пристрої - можуть погано масштабуватися
  • Серйозний контент - фінанси, медицина, юриспруденція

⚡ Оптимізація продуктивності

🚀 Поради з оптимізації веб-шрифтів:
  • font-display: swap; - показувати fallback шрифт під час завантаження
  • Preload критичних шрифтів: <link rel="preload" as="font">
  • Завантажуйте тільки потрібні ваги - не завантажуйте всі 9 вагів
  • Використовуйте локальні шрифти: local() в @font-face
  • WOFF2 формат - найкраще стиснення
  • Підтримка змінних шрифтів - один файл замість багатьох
/* Приклад оптимізованого підключення через @font-face */ @font-face { font-family: 'Inter'; src: local('Inter'), url('inter.woff2') format('woff2'), url('inter.woff') format('woff'); font-display: swap; }
<!-- Або через Google Fonts з font-display --> <link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;600&display=swap" rel="stylesheet">

🎯 Кращі практики

⚠️ Часті помилки в типографіці:
  • Занадто маленький шрифт - менше 16px на мобільних
  • Занадто маленький line-height - менше 1.2
  • Занадто довгі рядки - понад 80 символів
  • Низький контраст - менше 4.5:1
  • Багато різних шрифтів - більше 2-3 на сторінці
  • Ігнорування мобільних пристроїв
✅ Чек-лист хорошої типографіки:
  • ✓ Розмір основного тексту мінімум 16px
  • ✓ Line-height між 1.4-1.6 для тексту
  • ✓ Довжина рядка 45-75 символів
  • ✓ Контраст мінімум 4.5:1
  • ✓ Логічна ієрархія заголовків
  • ✓ Адаптивність на всіх пристроях
  • ✓ Швидке завантаження шрифтів
