Формати кольорів CSS

CSS підтримує кілька способів задання кольорів. Кожен має свої переваги та застосування:

#️⃣

HEX кольори

Шістнадцятковий формат

  • #ff0000
  • #00ff00
  • #0000ff
  • #333
🔴

RGB кольори

Red, Green, Blue

  • rgb(255, 0, 0)
  • rgb(0, 255, 0)
  • rgb(0, 0, 255)
  • rgba(255, 0, 0, 0.5)
🎨

HSL кольори

Hue, Saturation, Lightness

  • hsl(0, 100%, 50%)
  • hsl(120, 100%, 50%)
  • hsl(240, 100%, 50%)
  • hsla(0, 100%, 50%, 0.5)
📝

Іменовані кольори

Предвизначені назви

  • red
  • blue
  • green
  • transparent

Порівняння колірних моделей

Формат Синтаксис Переваги Недоліки Використання
HEX #RRGGBB або #RGB Компактний, популярний Важко читати людині Дизайн-системи, константи
RGB rgb(r, g, b) Зрозумілий, точний Довший запис Програмна генерація
RGBA rgba(r, g, b, a) Підтримка прозорості Довший запис Оверлеї, модальні вікна
HSL hsl(h, s%, l%) Інтуїтивний для дизайнерів Менш популярний Створення палітр
Named red, blue, etc. Читабельний Обмежений вибір Прототипування, базові кольори
💡 Коли використовувати кожен формат:
  • HEX - для брендингових кольорів та дизайн-системи
  • RGB/RGBA - для програмної генерації кольорів
  • HSL/HSLA - для створення палітр та варіацій
  • Named - для швидкого прототипування

♿ Перевірка доступності

Інтерактивна перевірка контрастності

Тестуйте свої кольори на відповідність стандартам WCAG:

Приклад тексту

Цей текст демонструє контраст між вибраними кольорами. Великий текст (18pt+ або 14pt+ жирний) має менші вимоги до контрастності.

Малий текст повинен мати вищий контраст
Контрастність: 21:1
AA Normal: ✅ Pass
AA Large: ✅ Pass
AAA Normal: ✅ Pass

Приклади контрастності

Чорний на білому
21:1 ✅ AAA
Білий на синьому
4.6:1 ✅ AA
Темний на жовтому
12.8:1 ✅ AAA
Білий на червоному
3.9:1 ⚠️ AA Large
Жовтий на блакитному
3.2:1 ⚠️ AA Large
Рожевий на фіолетовому
2.1:1 ❌ Fail
📏 Стандарти контрастності WCAG:
  • AA Normal: мінімум 4.5:1 для звичайного тексту
  • AA Large: мінімум 3:1 для великого тексту (18pt+ або 14pt+ жирний)
  • AAA Normal: мінімум 7:1 для найвищого рівня доступності
  • AAA Large: мінімум 4.5:1 для великого тексту

🛠️ Інструменти для перевірки доступності

Онлайн інструменти:
  • WebAIM Contrast Checker - найпопулярніший інструмент
  • Colour Contrast Analyser (CCA) - десктопна програма
  • Stark - плагін для Figma/Sketch
  • Accessible Colors - підбір доступних кольорів
Браузерні інструменти:
  • Chrome DevTools - Elements → Styles → Color picker → Contrast ratio
  • Firefox DevTools - Inspector → Accessibility panel
  • Lighthouse - автоматичний аудит доступності
Як перевіряти:
  1. Відкрийте DevTools (F12)
  2. Виберіть елемент з текстом
  3. У панелі Styles клацніть на квадратик кольору
  4. Браузер покаже контрастність автоматично
  5. Зелена галочка = відповідає стандартам
💡 Поради для кращої доступності:
  • Тестуйте кольори на різних пристроях і при різному освітленні
  • Не покладайтеся тільки на колір для передачі інформації
  • Додавайте іконки або підкреслення до кольорового кодування
  • Тестуйте з симуляторами колірної сліпоти
  • Враховуйте, що контраст може змінюватися на різних екранах

Практичні приклади використання

CSS
/* CSS змінні для колірної схеми */ :root { --primary-color: #667eea; --secondary-color: #764ba2; --success-color: #4caf50; --warning-color: #ff9800; --error-color: #f44336; --text-color: #2d3748; --bg-color: #ffffff; --shadow-color: rgba(0, 0, 0, 0.1); } /* Використання в компонентах */ .button { background: linear-gradient(135deg, var(--primary-color), var(--secondary-color)); color: white; border: none; padding: 0.75rem 1.5rem; border-radius: 8px; box-shadow: 0 4px 8px var(--shadow-color); } .success-message { background: hsla(122, 39%, 49%, 0.1); border-left: 4px solid var(--success-color); color: var(--success-color); padding: 1rem; } /* Dark mode підтримка */ @media (prefers-color-scheme: dark) { :root { --text-color: #e2e8f0; --bg-color: #1a202c; --shadow-color: rgba(0, 0, 0, 0.3); } }
🎯 Кращі практики роботи з кольорами:
  • Використовуйте CSS змінні для колірної схеми
  • Тестуйте контрастність для доступності
  • HSL зручний для створення варіацій одного кольору
  • RGBA/HSLA для напівпрозорих елементів
  • Підтримуйте dark mode через CSS змінні
