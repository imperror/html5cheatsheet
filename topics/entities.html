🔣 Спеціальні символи та HTML Entities

Вивчіть як правильно використовувати спеціальні символи в HTML документах

❓ Що таке HTML Entities?

HTML Entities - це спеціальні коди для відображення символів, які мають особливе значення в HTML або не можуть бути безпосередньо введені на клавіатурі.

💡 Чому потрібні Entities?

1. Зарезервовані символи: <, >, & мають спеціальне значення в HTML

2. Спеціальні символи: математичні, валютні, стрілки

3. Неломні пробіли: для точного контролю відстаней

4. Юнікод символи: емоджі, іноземні алфавіти

📝 Формати написання:

Іменовані: &lt; &gt; &amp;

Числові (десятичні): &#60; &#62; &#38;

Числові (шістнадцяткові): &#x3C; &#x3E; &#x26;

⚠️ Зарезервовані символи

Обов'язкові для екранування в HTML контенті

Символ HTML Entity Числовий код Опис Приклад використання
< &lt; &#60; Менше ніж (відкриваючий тег) Показати HTML код: &lt;div&gt;
> &gt; &#62; Більше ніж (закриваючий тег) Показати HTML код: &lt;/div&gt;
& &amp; &#38; Амперсанд (початок entity) Компанія Smith &amp; Co
" &quot; &#34; Подвійні лапки В атрибутах: title=&quot;Текст&quot;
' &apos; &#39; Одинарні лапки В тексті: It&apos;s working

⚠️ Важливо:

Завжди екрануйте <, >, & у текстовому контенті, щоб уникнути проблем з парсингом HTML.

📏 Спеціальні пробіли

Для точного контролю відстаней та типографії

Тип пробілу HTML Entity Ширина Опис та використання
Неломний пробіл &nbsp; Стандартний пробіл Запобігає переносу рядка між словами
En пробіл &ensp; Половина em (ширина літери N) Середній пробіл для типографії
Em пробіл &emsp; Ширина літери M Широкий пробіл для відступів
Тонкий пробіл &thinsp; 1/6 від em Вузький пробіл для точної типографії
Пробіл нульової ширини &#8203; 0 Невидимий символ для контролю переносів

✅ Кращі практики

✅ Рекомендації:

  • Завжди екрануйте <, >, & у текстовому контенті
  • Використовуйте &nbsp; для неломних пробілів
  • Віддавайте перевагу іменованим entities (&copy; замість &#169;)
  • Для емоджі використовуйте пряме введення в UTF-8
  • Перевіряйте підтримку символів у різних браузерах

⚠️ Обережно:

  • Не зловживайте неломними пробілами для форматування
  • Перевіряйте коректність відображення на мобільних пристроях
  • Деякі старі браузери можуть не підтримувати нові символи
  • Використовуйте fallback для критично важливих символів
