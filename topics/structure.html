📄 Структура HTML документа

Розуміння базової структури HTML документа - фундамент для створення якісних веб-сторінок

🏗️ Анатомія HTML документа

Кожен HTML документ має чітко визначену структуру, яка складається з обов'язкових та опціональних елементів. Розглянемо кожен компонент детально:

<!DOCTYPE html> Оголошення типу документа
<html> Кореневий елемент
<head> Мета-інформація
<meta> Кодування та мета-дані
<title> Заголовок документа
<body> Видимий контент
<header> Заголовок сторінки
<main> Основний контент
<footer> Нижній колонтитул

📋 Повна структура

<!DOCTYPE html>
<html lang="uk">
<head>
    <!-- Кодування символів -->
    <meta charset="UTF-8">
    
    <!-- Адаптивність -->
    <meta name="viewport"
          content="width=device-width,
                    initial-scale=1.0">
    
    <!-- SEO мета-теги -->
    <meta name="description"
          content="Опис сторінки">
    <meta name="keywords"
          content="HTML, веб-розробка">
    
    <!-- Заголовок -->
    <title>Моя веб-сторінка</title>
</head>
<body>
    <header>
        <h1>Заголовок сайту</h1>
        <nav>
            <!-- Навігація -->
        </nav>
    </header>
    
    <main>
        <article>
            <!-- Основний контент -->
        </article>
    </main>
    
    <footer>
        <!-- Нижній колонтитул -->
    </footer>
</body>
</html>

🔍 Детальний розбір елементів

<!DOCTYPE html> - Оголошення типу документа

Призначення: Вказує браузеру, що це HTML5 документ

Особливості:

  • Завжди перший рядок документа
  • Не є HTML тегом
  • Регістр не важливий
  • Обов'язковий для валідного HTML5

<html> - Кореневий елемент

Призначення: Контейнер для всього HTML контенту

Важливі атрибути:

  • lang="uk" - мова документа (для SEO та доступності)
  • dir="ltr" - напрямок тексту (зліва направо)

<head> - Мета-інформація

Призначення: Містить інформацію про документ (не відображається)

Може містити:

  • <title> - заголовок документа
  • <meta> - мета-дані
  • <link> - зовнішні ресурси (CSS)
  • <script> - JavaScript код
  • <style> - CSS стилі

<body> - Тіло документа

Призначення: Містить весь видимий контент сторінки

Семантичні елементи в body:

  • <header> - заголовок сторінки/розділу
  • <nav> - навігаційні елементи
  • <main> - основний контент
  • <article> - самостійний контент
  • <section> - розділи контенту
  • <aside> - бічна панель
  • <footer> - нижній колонтитул

✅ Кращі практики

🎯 SEO оптимізація:
  • Завжди вказуйте мову документа в атрибуті lang
  • Використовуйте описові <title> (50-60 символів)
  • Додавайте мета-опис для кращої індексації
📱 Адаптивність:
  • Обов'язково додавайте viewport мета-тег
  • Використовуйте семантичні HTML5 елементи
  • Дотримуйтесь правильної вкладеності тегів
♿ Доступність:
  • Використовуйте правильну ієрархію заголовків
  • Додавайте alt атрибути до зображень
  • Забезпечуйте логічну структуру документа
