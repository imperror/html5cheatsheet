Що таке CSS?

CSS (Cascading Style Sheets) - це мова стилів, яка використовується для опису зовнішнього вигляду документів HTML. Якщо HTML - це структура вашої веб-сторінки, то CSS - це її дизайн і презентація.

💡 Аналогія: Уявіть будинок - HTML це фундамент, стіни і каркас, а CSS це фарба, меблі, декор і все те, що робить будинок красивим і зручним.

Переваги CSS

✅ Основні переваги:
  • Розділення контенту і дизайну - HTML для структури, CSS для стилів
  • Багаторазове використання - один CSS файл для багатьох сторінок
  • Швидкість завантаження - зменшення розміру HTML файлів
  • Легке оновлення - зміна дизайну без зміни HTML
  • Доступність - кращий досвід для людей з обмеженими можливостями

Базовий синтаксис CSS

CSS складається з правил, кожне правило містить селектор і блок оголошень:

/* Базова структура CSS правила */ селектор { властивість: значення; інша-властивість: інше-значення; } /* Приклад реального CSS */ h1 { color: blue; font-size: 24px; text-align: center; } p { color: #333; line-height: 1.6; margin: 1rem 0; }
⚠️ Важливо: Кожне оголошення CSS повинно закінчуватися крапкою з комою ;
