🌊 CSS Float та Clear

Обтікання елементів та керування потоком контенту

🎯 Основи Float та Clear

Float та Clear - це класичні CSS властивості для створення макетів та обтікання елементів:

  • float - робить елемент "плаваючим", дозволяючи іншому контенту обтікати його
  • clear - забороняє елементу обтікати плаваючі елементи з певного боку
  • clearfix - техніка для вирішення проблеми "схлопнутих" контейнерів
📌 Важливо: Float в сучасному веб-розробці здебільшого замінений на Flexbox та Grid. Однак розуміння float залишається корисним для підтримки старого коду та специфічних задач.

🌊 Властивість Float

float: none
element { float: none; }
Значення за замовчуванням. Елемент не плаває та залишається в нормальному потоці документа.
float: left
element { float: left; }
Елемент прилипає до лівого краю контейнера. Інший контент обтікає його справа.
float: right
element { float: right; }
Елемент прилипає до правого краю контейнера. Інший контент обтікає його зліва.

🚫 Властивість Clear

Clear забороняє елементу обтікати плаваючі елементи з вказаного боку:

clear: none
element { clear: none; }
Значення за замовчуванням. Дозволяє обтікання з усіх боків.
clear: left
element { clear: left; }
Забороняє обтікання лівих плаваючих елементів. Елемент спуститься нижче за них.
clear: right
element { clear: right; }
Забороняє обтікання правих плаваючих елементів.
clear: both
element { clear: both; }
Забороняє обтікання плаваючих елементів з обох боків. Найчастіше використовується.

💡 Практичні приклади

📰 Обтікання зображення текстом

/* Класичне обтікання зображення */ .article-image { float: left; margin: 0 1rem 1rem 0; border-radius: 8px; } .article-text { text-align: justify; line-height: 1.6; }

📋 Простий макет з колонками

/* Двоколонний макет */ .sidebar { float: left; width: 25%; padding: 1rem; } .main-content { float: right; width: 70%; padding: 1rem; } .container { width: 100%; overflow: auto; /* Clearfix */ }

🎛️ Навігаційне меню

/* Горизонтальне меню */ .nav-item { float: left; margin-right: 1rem; } .nav-item:last-child { float: right; /* Останній пункт вправо */ } .nav-container::after { content: ""; clear: both; display: table; }

🔄 Альтернативи з Flexbox

/* Сучасна альтернativa Float */ /* Замість float: left + float: right */ .nav-container { display: flex; justify-content: space-between; } /* Замість складного clearfix */ .card-container { display: flex; flex-wrap: wrap; gap: 1rem; } /* Замість обтікання зображення */ .article { display: flex; gap: 1rem; }

🎯 Найкращі практики

✅ Коли використовувати Float:
  • Обтікання зображень текстом
  • Створення "обтічних" елементів в контенті
  • Підтримка дуже старих браузерів
  • Робота зі старим кодом
⚠️ Проблеми Float:
  • Складність у створенні макетів
  • Проблема схлопнутих контейнерів
  • Складність вертикального вирівнювання
  • Проблеми з відзивчивим дизайном
  • Потреба в clearfix хаках
🚀 Сучасні рекомендації:
  • Використовуйте Flexbox для одновимірних макетів
  • Використовуйте Grid для двовимірних макетів
  • Float залишайте тільки для обтікання контенту
  • Вивчайте сучасні техніки CSS
