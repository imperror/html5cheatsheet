📱 Responsive Design

Адаптивний дизайн для всіх пристроїв та екранів

🎯 Основи Responsive Design

Responsive Design - це підхід до веб-дизайну, який забезпечує оптимальне відображення сайту на всіх пристроях:

  • Гнучка сітка - макет адаптується до розміру екрана
  • Flexible Images - зображення масштабуються пропорційно
  • Media Queries - різні стилі для різних екранів
  • Mobile-First - спочатку для мобільних, потім для більших екранів
  • Viewport Meta Tag - правильне масштабування на мобільних
/* Обов'язковий viewport meta tag в HTML */ <meta name="viewport" content="width=device-width, initial-scale=1.0"> /* Базові responsive стилі */ * { box-sizing: border-box; } img { max-width: 100%; height: auto; } .container { max-width: 1200px; margin: 0 auto; padding: 0 1rem; }

📱

🔍 Media Queries

Media Queries дозволяють застосовувати різні стилі залежно від характеристик пристрою:

Основний синтаксис

/* Базовий синтаксис media query */ @media (max-width: 768px) { /* Стилі для екранів до 768px */ .container { padding: 1rem; } } @media (min-width: 769px) and (max-width: 1024px) { /* Стилі для екранів від 769px до 1024px */ .container { padding: 2rem; } } /* Mobile-first підхід */ .container { padding: 1rem; /* Мобільний по замовчуванню */ } @media (min-width: 768px) { .container { padding: 2rem; /* Планшет і більше */ } } @media (min-width: 1200px) { .container { padding: 3rem; /* Десктоп */ } }

Різні типи медіа-запитів

/* За шириною екрана */ @media (max-width: 768px) { /* До 768px */ } @media (min-width: 769px) { /* Від 769px */ } @media (width: 768px) { /* Точно 768px */ } /* За висотою екрана */ @media (max-height: 600px) { /* Низькі екрани */ } @media (min-height: 800px) { /* Високі екрани */ } /* За орієнтацією */ @media (orientation: portrait) { /* Портретна */ } @media (orientation: landscape) { /* Альбомна */ } /* За роздільною здатністю */ @media (min-resolution: 300dpi) { /* Retina екрани */ } @media (-webkit-min-device-pixel-ratio: 2) { /* Retina */ } /* За типом пристрою */ @media screen { /* Екрани */ } @media print { /* Принтери */ } /* Комбінування умов */ @media screen and (max-width: 768px) and (orientation: portrait) { /* Мобільні екрани в портретній орієнтації */ }

📏 Адаптивні одиниці виміру

Використовуйте правильні одиниці для створення гнучкого дизайну:

vw / vh
Відсоток від ширини/висоти viewport
%
Відсоток від батьківського елемента
rem
Відносно кореневого елемента
em
Відносно батьківського елемента
fr
Частка доступного простору (Grid)
clamp()
Мін, бажане та макс значення
/* Адаптивні одиниці в дії */ /* Viewport units */ .hero { height: 100vh; /* Висота екрана */ width: 100vw; /* Ширина екрана */ } /* Відсотки для контейнерів */ .container { width: 90%; /* 90% від батьківського елемента */ max-width: 1200px; } /* rem для типографіки */ h1 { font-size: 2rem; /* 32px якщо base 16px */ } /* clamp() для адаптивного тексту */ h1 { font-size: clamp(1.5rem, 4vw, 3rem); /* Мін 1.5rem, бажане 4vw, макс 3rem */ } /* Адаптивні відступи */ .section { padding: clamp(1rem, 5vw, 4rem); }

✅ Кращі практики

💡 Рекомендації:
  • Mobile-First: Почніть з мобільної версії, потім розширюйте
  • Тестуйте на реальних пристроях: Емулятори не завжди точні
  • Optimized Images: Використовуйте WebP, правильні розміри
  • Touch-Friendly: Кнопки мінімум 44px для пальців
  • Fast Loading: Мобільний інтернет може бути повільним
  • Content Priority: Найважливіший контент першим
⚠️ Поширені помилки:
  • Забувати про viewport meta tag
  • Використовувати фіксовані одиниці (px) замість адаптивних
  • Ігнорувати альбомну орієнтацію на мобільних
  • Занадто маленькі кнопки для сенсорних екранів
  • Надто багато контенту на мобільних
  • Не тестувати на реальних пристроях

🎯 Чек-лист responsive дизайну

/* Обов'язкові елементи */ 1. ✅ Viewport meta tag <meta name="viewport" content="width=device-width, initial-scale=1.0"> 2. ✅ Box-sizing для всіх елементів *, *::before, *::after { box-sizing: border-box; } 3. ✅ Responsive images img { max-width: 100%; height: auto; } 4. ✅ Mobile-first media queries @media (min-width: 768px) { /* Планшет+ */ } 5. ✅ Flexible units (%, rem, vw, vh) .container { width: 90%; max-width: 1200px; } 6. ✅ Touch targets 44px+ button { min-height: 44px; min-width: 44px; } 7. ✅ Fast loading (<3 секунди) Оптимізовані зображення, мінімум HTTP запитів 8. ✅ Content hierarchy Найважливіший контент першим на мобільних
