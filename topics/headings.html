📝 Заголовки в HTML5

Вивчіть ієрархію заголовків h1-h6 та кращі практики їх використання для SEO та доступності

🏗️ Ієрархія заголовків

HTML надає шість рівнів заголовків: від h1 (найважливіший) до h6 (найменш важливий). Правильна ієрархія заголовків є ключовою для SEO та доступності.

h1

Головний заголовок сторінки

h2

Заголовок розділу

h3

Підзаголовок розділу

h4

Заголовок підрозділу

h5
Заголовок п'ятого рівня
h6
Заголовок шостого рівня

💻 Приклад коду

<!-- Основна структура сторінки -->
<h1>Путівник по HTML5</h1>

<h2>Розділ 1: Основи</h2>
<h3>1.1 Що таке HTML?</h3>
<h4>1.1.1 Історія HTML</h4>
<h4>1.1.2 Версії HTML</h4>

<h3>1.2 Структура документа</h3>
<h4>1.2.1 Елемент DOCTYPE</h4>
<h4>1.2.2 Елементи head та body</h4>

<h2>Розділ 2: Семантика</h2>
<h3>2.1 Семантичні елементи</h3>

<!-- Важливо: не пропускайте рівні! -->

✅ Кращі практики

✅ Правильно:

  • Використовуйте тільки один h1 на сторінці
  • Дотримуйтесь логічної ієрархії (h1 → h2 → h3...)
  • Робіть заголовки описовими та змістовними
  • Використовуйте заголовки для структури, а не для стилізації

❌ Неправильно:

  • Не пропускайте рівні (h1 → h3 без h2)
  • Не використовуйте кілька h1 на одній сторінці
  • Не вибирайте заголовок тільки через розмір шрифту
  • Не залишайте заголовки порожніми

💡 Корисні поради:

  • Заголовки допомагають користувачам швидко сканувати контент
  • Пошукові системи використовують заголовки для розуміння структури
  • Екранні читачі використовують заголовки для навігації
  • Довжина заголовка h1 оптимально 20-70 символів для SEO

🚀 SEO та доступність

  • h1 - містить головні ключові слова сторінки
  • h2-h3 - підтримують тему та додають контекст
  • h4-h6 - для деталізації та уточнень
  • Ключові слова - розподіляйте природно по заголовках
  • Читабельність - заголовки мають бути зрозумілими людині
