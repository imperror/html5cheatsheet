📄 Параграфи та переноси рядків

Вивчіть основи роботи з текстом в HTML: параграфи, переноси рядків та форматування пробілів

📝 Елемент <p> - Параграфи

Параграф - це основний блоковий елемент для організації тексту в HTML. Кожен параграф автоматично відділяється від інших вертикальними відступами.

💻 HTML код:

<p>Це перший параграф тексту. Він містить кілька речень які пояснюють тему.</p>

<p>Це другий параграф. Між параграфами браузер автоматично додає відступи для кращої читабельності.</p>

<p>Третій параграф може містити <strong>важливий текст</strong> або <em>виділений текст</em>.</p>

🎨 Результат:

Це перший параграф тексту. Він містить кілька речень які пояснюють тему.

Це другий параграф. Між параграфами браузер автоматично додає відступи для кращої читабельності.

Третій параграф може містити важливий текст або виділений текст.

🎯 Особливості параграфів:

  • 📦 Блоковий елемент - займає всю доступну ширину
  • 📏 Автоматичні відступи зверху і знизу
  • 🔄 Може містити inline елементи та текст
  • 🚫 Не може містити інші блокові елементи

↩️ Елемент <br> - Перенос рядка

Елемент <br> створює принудовий перенос рядка без створення нового параграфа.

💻 HTML код:

<p>
    Перший рядок<br>
    Другий рядок<br>
    Третій рядок
</p>

<!-- Приклад для адреси -->
<address>
    вул. Хрещатик, 1<br>
    м. Київ, 01001<br>
    Україна
</address>

<!-- Поезія -->
<blockquote>
    Україно! Ти для мене свята,<br>
    Твоя доля — це моя дорога...<br>
    <cite>— Ліна Костенко</cite>
</blockquote>

➖ Елемент <hr> - Горизонтальна лінія

Елемент <hr> створює тематичний розрив між блоками контенту у вигляді горизонтальної лінії.

💻 HTML код:

<p>Перший розділ документа.</p>

<hr>

<p>Другий розділ документа після тематичного розриву.</p>

<!-- Розділення в статті -->
<article>
    <h2>Заголовок статті</h2>
    <p>Основний текст статті...</p>
    
    <hr>
    
    <footer>
        <p>Автор: Іван Петренко</p>
        <p>Дата: 15 січня 2024</p>
    </footer>
</article>

🎨 Результат:

Перший розділ документа.

Другий розділ документа після тематичного розриву.

Заголовок статті

Основний текст статті...

Автор: Іван Петренко

Дата: 15 січня 2024

📋 Елемент <pre> - Попередньо форматований текст

Елемент <pre> зберігає всі пробіли, табуляції та переноси рядків як вони є в коді.

💻 HTML код:

<pre>
    Цей текст
        зберігає
            всі    пробіли
                та переноси рядків
</pre>

<!-- Код програми -->
<pre><code>
function greet(name) {
    console.log("Привіт, " + name + "!");
}

greet("світ");
</code></pre>

<!-- ASCII арт -->
<pre>
  /\_/\  
 ( o.o ) 
  > ^ <
</pre>

🎨 Результат:

    Цей текст
        зберігає
            всі    пробіли
                та переноси рядків
function greet(name) {
    console.log("Привіт, " + name + "!");
}

greet("світ");
  /\_/\  
 ( o.o ) 
  > ^ <

🔤 Як браузер обробляє пробіли (Whitespace)

HTML схлопує множинні пробіли, табуляції та переноси рядків в один пробіл. Це називається whitespace collapsing.

Вхідний HTML код:
<p>Цей текст має
    багато     пробілів
        та   переносів</p>
Результат у браузері:
Цей текст має багато пробілів та переносів

🎯 Інтерактивна демонстрація пробілів

Звичайний текст без додаткових пробілів.

📊 Порівняння елементів

Елемент Призначення Тип Пробіли
<p> Параграф тексту Блоковий Схлопуються
<br> Перенос рядка Inline, void —
<hr> Тематичний розрив Блоковий, void —
<pre> Попередньо форматований Блоковий Зберігаються

✅ Кращі практики

✅ Рекомендації:

  • Використовуйте <p> для звичайних параграфів
  • <br> для адрес, віршів, коротких списків
  • <hr> для тематичних розривів
  • <pre> для коду та ASCII-арту
  • Робіть параграфи логічно завершеними

⚠️ Обережно:

  • Не використовуйте <br> для створення відступів
  • Уникайте багато <br> підряд
  • Не ставте блокові елементи в <p>
  • <pre> може псувати адаптивність

❌ Уникайте:

  • Порожніх параграфів <p></p>
  • Використання <br> замість CSS margin
  • Декоративного використання <hr>
  • Довгих рядків в <pre>

🎨 HTML сутності для пробілів

Спеціальні символи пробілів

📏 Типографічні одиниці:
em - ширина літери "M" у поточному шрифті (відносна одиниця)
en - половина em, приблизно ширина літери "N"
&nbsp; Нерозривний пробіл
&ensp; Пробіл en (половина em)
&emsp; Пробіл em (ширина літери M)
&thinsp; Тонкий пробіл (1/6 em)

Приклад:

<p>Word&nbsp;with&nbsp;non-breaking&nbsp;spaces</p>
<p>Thin&thinsp;space&thinsp;example</p>

Результат:

Word with non-breaking spaces

Thin space example
