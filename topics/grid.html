🔲 CSS Grid Layout

Потужна двовимірна система макетування для складних дизайнів

🎯 Основи CSS Grid

CSS Grid - це найпотужніша система макетування, яка дозволяє створювати складні двовимірні макети:

  • Grid Container - батьківський елемент з display: grid
  • Grid Items - дочірні елементи, що автоматично розміщуються в сітці
  • Grid Lines - лінії що розділяють сітку (вертикальні та горизонтальні)
  • Grid Tracks - простір між двома лініями (рядки та колонки)
  • Grid Areas - прямокутні області, що складаються з кількох клітинок
  • Grid Gap - відстань між клітинками сітки
/* Основний синтаксис */ .container { display: grid; /* Створює grid-контейнер */ grid-template-columns: 1fr 2fr 1fr; /* 3 колонки */ grid-template-rows: auto 1fr auto; /* 3 рядки */ gap: 1rem; /* Відступи між елементами */ }

📐 Властивості Grid Container

grid-template-columns
1fr 2fr | 200px auto | repeat(3, 1fr)
Визначає ширину та кількість колонок в сітці.
grid-template-rows
auto 1fr auto | 100px 200px | minmax(100px, auto)
Визначає висоту та кількість рядків в сітці.
grid-template-areas
"header header" "sidebar main" "footer footer"
Визначає іменовані області сітки для легкого позиціонування.
gap (grid-gap)
1rem | 10px 20px | 2rem 1rem
Задає відстань між рядками та колонками.
justify-items
start | center | end | stretch
Вирівнювання елементів всередині клітинок по горизонталі.
align-items
start | center | end | stretch
Вирівнювання елементів всередині клітинок по вертикалі.

📦 Властивості Grid Items

grid-column
1 / 3 | span 2 | 1 / -1
Позиціонування елемента по колонках (початок / кінець).
grid-row
2 / 4 | span 3 | 1 / -1
Позиціонування елемента по рядках (початок / кінець).
grid-area
header | 1 / 1 / 3 / 3 | span 2 / span 3
Скорочення для позиціонування або іменована область.
justify-self
start | center | end | stretch
Вирівнювання конкретного елемента по горизонталі.
align-self
start | center | end | stretch
Вирівнювання конкретного елемента по вертикалі.
/* Типові комбінації grid */ /* Елемент займає 2 колонки */ .wide { grid-column: span 2; } /* Елемент займає всю ширину */ .full-width { grid-column: 1 / -1; } /* Позиціонування за лініями */ .positioned { grid-column: 2 / 4; grid-row: 1 / 3; } /* Використання іменованих областей */ .header { grid-area: header; }

⚖️ Grid vs Flexbox: Коли що використовувати

Критерій CSS Grid Flexbox
Вимірність Двовимірний (рядки + колонки) Одновимірний (тільки рядок або колонка)
Кращий для Складні макети, сітки, dashboard Навігація, вирівнювання, прості макети
Підхід Layout-first (спочатку макет, потім контент) Content-first (контент визначає макет)
Розмір елементів Можна задавати точні позиції та розміри Елементи розподіляються автоматично
Підтримка браузерів IE11+ (з префіксами) IE10+ (з префіксами)
Вивчення Складніше, більше властивостей Простіше, менше властивостей
🎯 Рекомендації:
  • Використовуйте Grid для: макетів сторінок, галерей, dashboard, форм
  • Використовуйте Flexbox для: навігації, кнопок, центрування, карток в ряд
  • Комбінуйте: Grid для загального макету, Flexbox всередині Grid-елементів
  • Правило: якщо потрібні і рядки і колонки - Grid, тільки одне - Flexbox

🔧 Корисні функції та одиниці Grid

🔢 Функції Grid

/* repeat() - повторення значень */ grid-template-columns: repeat(3, 1fr); /* Замість 1fr 1fr 1fr */ grid-template-columns: repeat(auto-fit, minmax(200px, 1fr)); /* Адаптивні колонки */ /* minmax() - мінімальне та максимальне значення */ grid-template-columns: minmax(200px, 1fr) 300px; /* Мін 200px, макс весь простір */ grid-template-rows: minmax(100px, auto); /* Мін 100px, висота по контенту */ /* auto-fit vs auto-fill */ grid-template-columns: repeat(auto-fit, minmax(200px, 1fr)); /* Розтягує колонки */ grid-template-columns: repeat(auto-fill, minmax(200px, 1fr)); /* Зберігає розмір */ /* fit-content() - розмір по контенту з максимумом */ grid-template-columns: fit-content(200px) 1fr; /* Макс 200px або по контенту */

📏 Одиниці Grid

/* fr (fraction) - частки доступного простору */ grid-template-columns: 1fr 2fr 1fr; /* 1:2:1 пропорції */ grid-template-columns: 200px 1fr; /* 200px + решта простору */ /* auto - розмір по контенту */ grid-template-columns: auto 1fr auto; /* Контент - гнучкий - контент */ /* min-content, max-content */ grid-template-columns: min-content 1fr; /* Мінімальна ширина контенту */ grid-template-columns: max-content 1fr; /* Максимальна ширина контенту */

🎯 Практичні поради

💡 Поради по Grid:
  • Використовуйте gap замість margin для відступів
  • repeat(auto-fit, minmax(200px, 1fr)) - найкращий спосіб для адаптивності
  • Іменовані лінії та області роблять код більш читабельним
  • grid-auto-rows: minmax(150px, auto) для рівної висоти рядків
  • Завжди тестуйте на різних розмірах контенту
  • Grid DevTools в браузері - ваш найкращий друг для налагодження
