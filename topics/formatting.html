✨ Форматування тексту

Вивчіть семантичні та візуальні елементи для правильного форматування тексту в HTML

🎯 Семантичне vs Візуальне форматування

HTML надає два типи елементів форматування: семантичні (передають значення) та візуальні (змінюють тільки вигляд).

✅ Семантичні елементи

Передають значення тексту для браузерів, пошукових систем та екранних читачів.

  • <strong> - важливий текст
  • <em> - наголошений текст
  • <mark> - виділений текст
  • <del> - видалений текст
  • <ins> - доданий текст

⚠️ Візуальні елементи

Змінюють тільки вигляд без передачі семантичного значення.

  • <b> - жирний текст
  • <i> - курсивний текст
  • <u> - підкреслений текст
  • <s> - закреслений текст
  • <small> - малий текст

💡 Коли використовувати?

Семантичні елементи - коли текст має особливе значення (важливість, наголос, виділення)

Візуальні елементи - коли потрібно тільки змінити вигляд без додавання значення

🔥 Основні семантичні елементи

Елементи, які передають значення та важливість тексту

Важливий текст <strong>
<p>Це <strong>дуже важлива</strong> інформація!</p>

Це дуже важлива інформація!

Використання: для тексту високої важливості

Наголошений текст <em>
<p>Я <em>справді</em> люблю HTML!</p>

Я справді люблю HTML!

Використання: для логічного наголосу

Виділений текст <mark>
<p>Знайдено <mark>HTML5</mark> у тексті</p>

Знайдено HTML5 у тексті

Використання: для підсвічування в результатах пошуку

Видалений текст <del>
<p>Ціна: <del>100₴</del> 80₴</p>

Ціна: 100₴ 80₴

Використання: для показу видаленого контенту

Доданий текст <ins>
<p>Курс: <ins>HTML5 для початківців</ins></p>

Курс: HTML5 для початківців

Використання: для показу доданого контенту

Малий текст <small>
<p>Головний текст <small>(примітка)</small></p>

Головний текст (примітка)

Використання: для авторських прав, примітків

💻 Комп'ютерні та технічні елементи

Спеціальні елементи для коду, клавіш та технічної документації

Код у тексті <code>
<p>Використовуйте <code>console.log()</code> для виводу</p>

Використовуйте console.log() для виводу

Використання: для фрагментів коду в тексті

Клавіші <kbd>
<p>Натисніть <kbd>Ctrl</kbd> + <kbd>C</kbd></p>

Натисніть Ctrl + C

Використання: для комбінацій клавіш

Вивід програми <samp>
<p>Результат: <samp>Hello, World!</samp></p>

Результат: Hello, World!

Використання: для прикладів виводу

Змінні <var>
<p>Площа = <var>π</var> × <var>r</var><sup>2</sup></p>

Площа = π × r2

Використання: для математичних змінних

Надрядковий текст <sup>
<p>H<sub>2</sub>O та E=mc<sup>2</sup></p>

H2O та E=mc2

Використання: для ступенів, індексів

Абревіатури <abbr>
<p><abbr title="Україна">UA</abbr> - моя країна</p>

UA - моя країна

Використання: для скорочень з поясненням

🎮 Інтерактивна демонстрація

✅ Кращі практики

✅ Рекомендації:

  • Використовуйте семантичні елементи для передачі значення
  • <strong> для важливого тексту, а не тільки жирного
  • <em> для наголосу, а не тільки курсиву
  • <code> для фрагментів коду в тексті
  • <kbd> для клавіш та комбінацій
  • Додавайте title до <abbr> для пояснення

⚠️ Обережно:

  • Не зловживайте візуальними елементами (b, i, u)
  • Уникайте <u> - може плутатися з посиланнями
  • Не використовуйте <small> тільки для зменшення розміру
  • Перевіряйте контрастність для <mark>

❌ Уникайте:

  • Використання <b> замість <strong>
  • Використання <i> замість <em>
  • Надмірного форматування в одному тексті
  • Форматування замість CSS для стилізації
